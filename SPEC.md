# Hockey Pattern Predictor - Спецификация интерфейса и логики

## 1. Общая концепция

Веб-приложение для анализа хоккейных матчей на основе паттернов. Сочетает машинные прогнозы (ML модель) с ручным отбором пользователя.

### Цель
- Показать предстоящие матчи с визуализацией паттернов
- Помочь пользователю быстро оценить "силу" сигнала прерывания
- Предоставить детальную информацию для ручного анализа

## 3. Логика паттернов

### 3.1 Типы паттернов

| Паттерн | Описание | Критическая длина |
|---------|----------|-------------------|
| **Серия побед/поражений** | Последовательность W или L | ≥5 |
| **Домашняя серия** | Серия дома | ≥5 |
| **Гостевая серия** | Серия в гостях | ≥5 |
| **Чередование** | Паттерн WLWLWL | ≥5 |
| **Перерост** | Превышение средней длины серии | >3σ |

### 3.2 Расчёт Score (силы сигнала)

```
Score = сумма факторов:
  +1 если есть критическая серия (≥5)
  +1 если есть критическое чередование (≥5)
  +1 если синергия ≥2 (несколько паттернов)
  +1 если перерост серии
  +1 если обе команды имеют критические паттерны
  +1 если паттерны противоречат (указывают на разные исходы)
```

**Интерпретация:**
- Score 0-2: Слабый сигнал
- Score 3-4: Средний сигнал
- Score 5-6: Сильный сигнал

## 4. ML модель

### 4.1 Текущая модель

- **Алгоритм:** Random Forest с калибровкой вероятностей
- **Признаки:** 81 (серии, чередования, синергия, домашний/гостевой)
- **Целевая переменная:** Прерывание (1) / Продолжение (0)
- **Точность:** 78%
- **Recall прерываний:** 91%

### 4.2 Ключевые признаки

1. `home_overall_win_streak` (16.3%)
2. `home_any_critical` (12.8%)
3. `critical_synergy_home` (6.0%)
4. `home_total_critical_patterns` (6.0%)
5. `home_total_critical` (5.8%)

### 4.3 Интерпретация прогноза

```
Прогноз модели:
  - break_probability: вероятность прерывания серии
  - continue_probability: вероятность продолжения
  - recommendation: ПРЕРЫВАНИЕ / ПРОДОЛЖЕНИЕ
```

## 5. Интеграция коэффициентов

### 5.1 Источники данных

| Лига | API | Статус |
|------|-----|--------|
| NHL | The Odds API | ✅ Работает |
| SHL, Liiga | The Odds API | ✅ Работает |
| KHL, DEL | - | ❌ Нет данных |

### 5.2 Расчёт EV (Expected Value)

```python
# Логика EV расчёта
if серия_побед:
    bet_on = противник  # ставим против серии
    odds = коэф_противника
else:
    bet_on = команда_с_серией
    odds = коэф_команды

EV = (break_prob × (odds - 1)) - (1 - break_prob)

# EV > 0 = выгодная ставка
```
