# Теория Циклического Прерывания Паттернов (ЦПП)

Автоматизированная система для анализа спортивных матчей, выявления паттернов и формирования прогнозов.

## Основная идея

Теория ЦПП основывается на предположении, что в спортивных результатах существуют определённые закономерности, которые периодически нарушаются.

## Основные аспекты

1. **Выявление устойчивых паттернов** в результатах команд
2. **Определение факторов**, влияющих на прерывание этих паттернов
3. **Оценка вероятности** прерывания паттерна в конкретном матче

## Гипотеза

Используя исторические данные и контекстуальную информацию, мы можем создать модель, способную с определённой точностью предсказывать моменты, когда устоявшийся паттерн результатов команды может быть нарушен.

## Критические длины паттернов

| Тип паттерна | Критическая длина | Описание |
|--------------|-------------------|----------|
| Общая серия побед/поражений | 5-7 матчей | Базовый показатель формы |
| Домашняя серия | 4-6 матчей | С учётом фактора площадки |
| Гостевая серия | 3-5 матчей | Усталость от переездов |
| Бинарное чередование (W-L) | 6-8 элементов | Высокая предсказуемость |
| Тернарное чередование (W-D-L) | 6-9 элементов | Редко достигается |
| Паттерн "два-один" (W-W-L) | 6 элементов | Команды среднего уровня |
| Личные встречи | 3-5 матчей | Высокая прогностическая ценность |

## Ключевые принципы

### 1. Принцип критической длины
Когда паттерн достигает критической длины, вероятность его прерывания резко возрастает.

### 2. Принцип синергии
Когда несколько паттернов достигают критической длины одновременно, это значительно увеличивает вероятность прерывания всех паттернов.

### 3. Принцип противоречия
Если несколько паттернов указывают в одном направлении и все достигли критической длины, наиболее вероятен **противоположный** результат.

### 4. Принцип антихрупкости
Система использует неожиданные результаты для улучшения прогнозов — каждое прерывание паттерна добавляет данные для более точного определения критических точек.

## Формула расчёта ценности ставки (EV)

```
EV = (вероятность_прерывания × коэффициент) - 1
```

Примеры:
- EV = (0.25 × 6.20) - 1 = **+55%** — отличная ценность
- EV = (0.60 × 2.10) - 1 = **+26%** — хорошая ценность
- EV = (0.40 × 2.00) - 1 = **-20%** — нет ценности

## Контекстуальные факторы

Помимо паттернов, учитываются:

1. **Мотивация** — борьба за плей-офф, принципиальные матчи
2. **Физическое состояние** — травмы, усталость, плотный график
3. **Принципиальные дерби** — особая эмоциональная составляющая
4. **Турнирное положение** — влияет на мотивацию команд

## Цели системы

1. Создание модели машинного обучения для выявления паттернов
2. Разработка алгоритма оценки вероятности прерывания
3. Интеграция с системой уведомлений для выгодных ставок
4. Достижение положительного ROI при реальных ставках
