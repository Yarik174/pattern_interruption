## Ключевые концепции

### Критическая длина
Когда паттерн достигает определённой длины, вероятность его прерывания резко возрастает.

### Синергия
Несколько критических паттернов, указывающих в одном направлении, усиливают сигнал.

### Противоход
Когда ВСЕ паттерны указывают на один результат и все достигли критической длины — жди обратного.

### EV (Expected Value)
```
EV = (вероятность × коэффициент) - 1
```
Положительный EV = выгодная ставка.
