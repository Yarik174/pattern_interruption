## Типы ситуаций

### Высокая уверенность (8-9/10)
- **Пример 7 (Эдмонтон vs Калгари):** Тройное прерывание — все паттерны указывают в одну сторону
- **Пример 2 (Даллас vs Вегас):** Чёткое чередование с критической длиной
- **Пример 4 (Флорида vs Айлендерс):** Тройная синергия с весовым преимуществом

### Средняя уверенность (7/10)
- **Пример 1, 3:** Двойная/тройная синергия критических паттернов
- **Пример 6:** Конфликт паттернов → ничья
- **Пример 9:** Противоход при совпадении паттернов

### Низкая уверенность (6/10)
- **Пример 5:** Только один критический паттерн
- **Пример 8:** Нет критических паттернов, только контекст

## Ключевые принципы из примеров

1. **Синергия усиливает сигнал** — несколько критических паттернов в одном направлении
2. **Противоход при совпадении** — когда всё указывает на один результат, жди обратного
3. **Контекст при отсутствии паттернов** — мотивация, травмы, дерби
4. **Конфликт → компромисс** — при противоречии паттернов возможна ничья
