## Правила ЦПП

### Типы паттернов и их прерывание

| Тип паттерна | Пример | При прерывании |
|--------------|--------|----------------|
| Серия (одинаковые результаты) | WWWWW или LLLLL | **Противоположный** результат |
| Чередование | WLWLWL или LWLWLW | **Повтор** последнего результата |

### Синергия и конфликт

- **Синергия:** несколько паттернов указывают в одном направлении → сильный сигнал → ставка
- **Конфликт:** паттерны указывают в разных направлениях → нет чёткого сигнала → пропуск

## Типы ситуаций

### Сильный сигнал (8/10)
- **Пример 1:** Тройная синергия серий → все указывают на одного победителя
- **Пример 3:** Двойная синергия серий → общая серия + H2H
- **Пример 7:** Синергия 2:1 чередований → Эдмонтон + H2H vs Калгари
- **Пример 9:** Двойная синергия чередований → оба паттерна за Сиэтл

### Средний сигнал (6-7/10)
- **Пример 2:** Один критический паттерн чередования
- **Пример 4:** Синергия 2:1 серий
- **Пример 5:** Один критический паттерн чередования

### Нет сигнала
- **Пример 6:** Конфликт паттернов → пропуск
- **Пример 8:** Нет критических паттернов → пропуск

## Ключевые принципы

1. **Серия прерывается противоположным результатом**
   - 5 побед подряд → следующий матч поражение
   - 5 поражений подряд → следующий матч победа

2. **Чередование прерывается повтором**
   - W-L-W-L-W-L, последний L → следующий тоже L
   - L-W-L-W-L-W, последний W → следующий тоже W

3. **Синергия усиливает сигнал**
   - Несколько критических паттернов указывают на один исход
   - Чем больше паттернов совпадают, тем выше уверенность

4. **Конфликт = пропуск**
   - Если паттерны противоречат друг другу — нет ставки
