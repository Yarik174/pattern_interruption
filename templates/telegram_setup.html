📝 Инструкция по созданию бота

1

Откройте BotFather в Telegram

Перейдите по ссылке t.me/BotFather или найдите @BotFather в поиске Telegram.

2

Создайте нового бота

Отправьте команду /newbot и следуйте инструкциям. Придумайте имя (например, "Hockey Predictor") и username (например, hockey_predictor_bot).

3

Скопируйте токен

BotFather пришлёт вам API токен вида 123456789:ABCdefGHIjklMNOpqrsTUVwxyz. Скопируйте его.

4

Получите Chat ID

Напишите своему боту любое сообщение, затем откройте:
https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
Найдите "chat":{"id": ЧИСЛО} — это ваш Chat ID.

⚙️ Сохранить настройки

Токен и Chat ID сохраняются как секреты в настройках проекта для безопасности.
